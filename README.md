# VK Album booking parser

Пользовательский скрипт для парсинга комментариев с "бронь" в альбомах ВКонтакте. Группирует результаты по пользователям и экспортирует в CSV.

## Установка

### Вариант 1: Установка в один клик

[![Установить](https://img.shields.io/badge/Установить-скрипт-brightgreen)](https://github.com/awaynell/vk-album-booking-parser/raw/main/vk-album-booking-parser.user.js)

> **Важно:** Убедитесь, что у вас установлен менеджер пользовательских скриптов:
>
> - [Tampermonkey](https://www.tampermonkey.net/) (рекомендуется)
> - [Violentmonkey](https://violentmonkey.github.io/)
> - [Greasemonkey](https://www.greasespot.net/)

### Вариант 2: Ручная установка

1. Установите менеджер пользовательских скриптов (Tampermonkey, Violentmonkey и т.д.)
2. Откройте файл `vk-album-booking-parser.user.js`
3. Скопируйте весь код
4. Откройте менеджер скриптов → "Создать новый скрипт"
5. Вставьте код и сохраните

## Использование

1. Откройте любой альбом ВКонтакте (`https://vk.com/album*`)
2. Нажмите кнопку **"Старт"** в правом нижнем углу
3. Скрипт автоматически пройдёт по всем фотографиям и найдёт комментарии с "бронь"
4. Результаты отобразятся в таблице, сгруппированные по пользователям
5. Нажмите **"CSV"** для экспорта данных

## Возможности

- Автоматический парсинг всех фотографий альбома
- Поиск комментариев с ключевым словом "бронь"
- Группировка результатов по пользователям
- Экспорт в CSV формат
- Подсветка найденных фотографий
