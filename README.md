# VK Album booking parser

Юзер-скрипт для ВК: проходит по фото в альбоме, ищет в комментариях **«бронь»** и собирает результат **по пользователям** (юзер → список фото). Можно выгрузить в **CSV**.

## Что нужно для работы

Поставьте менеджер юзер-скриптов:

- Tampermonkey (рекомендую)
- Violentmonkey
- Greasemonkey

## Как пользоваться

1. Откройте альбом ВК: `https://vk.com/album...`
2. Внизу справа появится панель — нажмите **Старт**
3. Готово: увидите таблицу “юзер → ссылки на фото”, можно скачать **CSV**
